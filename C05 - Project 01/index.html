Front-end web development, also known as client-side development is the practice of producing HTML, CSS and JavaScript for a website or Web Application so that a user can see and interact with them directly. The challenge associated with front end development is that the tools and techniques used to create thefront end of a website change constantly and so the developer needs to constantly be aware of how the field is developing.

The objective of designing a site to ensure that when the users open the site they see the information in a format that is easy to read and relevant. This is further complicated by the fact that the user now use a large variety of devices with varying screen sizes and resolutions thus forcing the designer to take into consideration these aspects when designing the site. They need to ensure that their site comes up correctly in different browsers (cross-brower), different operating systems (cross-platform) and different devices (cross-device), which require careful planning on the side of the developer.
